# Quickvee Help Assets

Public image asset repository for Quickvee Help Center, Confluence, and Quickvee Training embeds.

## CDN format

Use jsDelivr for stable direct image embeds:

```
https://cdn.jsdelivr.net/gh/quickveemktg/quickvee-help-assets@main/PATH/TO/IMAGE.png
```

Example:

```
https://cdn.jsdelivr.net/gh/quickveemktg/quickvee-help-assets@main/MM-287-01.png
```

## Important

Only upload non-sensitive, merchant-facing screenshots. Do not upload private customer data, internal-only credentials, unpublished roadmap content, or screenshots with real merchant/customer personal information.
